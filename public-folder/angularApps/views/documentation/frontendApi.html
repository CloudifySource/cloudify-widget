We recommend reading Mozilla's short documentation
on the topic. This page will summarize the message the widget receives and the messages it will post.

General

basically all the messages (posted or received) have the same structure.
{ "name" : event_name, "data" : data_object }
You can pass an object or a string, the widget will handle conversion to an object in case it is a string.
The widget might pass a string or an object.
We hold the right to change this at any given moment.
We recommend using a safe code.
If you are using angular you can simply use
angular.toJson(event.data);
and if you do not have angularjs you can always do
if ( typeof(event.data) === 'string' ){ event.data = JSON.parse(event.data); }
